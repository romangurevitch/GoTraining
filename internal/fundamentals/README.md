# Module 1: Cloud API Engineering Principles

> Content for this module is under development.

This module covers the foundational concepts for building production-ready APIs and platform tools.

## Topics

- **API Fundamentals** — REST vs RPC, idempotency, statelessness, contract-first vs code-first design
- **Security & Observability** — AuthN/AuthZ, policy as code, structured logging with `slog`, distributed tracing
- **API Lifecycle & Deployment** — versioning (v1/v2), graceful sunsetting, containerisation, cloud deployment
- **The Agentic Future** — designing APIs for AI consumption, Model Context Protocol (MCP), tool discovery
